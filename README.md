# level-1-wizard

1. Clone repo
    `git clone https://github.com/zjtheilen/level-1-wizard.git`

2. Create a virtual environment
    `python -m venv venv`

3. Enter virtual environment
    `.\.venv\Scripts\activate`

4. Install required libraries
    `pip install -r .\requirements.txt`

5. Run app
    `python app.py`
